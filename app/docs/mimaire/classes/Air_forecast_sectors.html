get_data_by_model

get_data_by_model() : \JSON

Datos de variable de calidad del aire para el Receptor de un Sector. Carga los datos de pronóstico de la variable en una fecha y hora determinadas. Se utiliza en la vista principal del módulo de Pronósticos via Ajax que se ejecuta mediante el evento on_change de los selectores de variables y receptor, de las secciones de Modelo Estadístico y Modelo Neuronal, para actualizar los datos de los gráficos y calheatmap.
